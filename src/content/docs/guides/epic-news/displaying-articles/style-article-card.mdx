### Capturing an item from an array

Notice how we captured the first item from the `mockArticles` array with square brackets `[]` and the index `0`?

```tsx
const firstArticle = mockArticles[0]
```

If we wanted to access the second item in the array, we would use `mockArticles[1]`, the third item would be `mockArticles[2]` and so on.

This is how you access an individual array item in JavaScript.

:::

## Display the first article

Finally, let's display the first article's `title`, `image` and `category` on the page:

```diff lang="tsx" title="app/routes/news.$category.tsx"
export default function NewsCategoryPage() {
	const { category, mockArticles } = useLoaderData<typeof loader>()

	console.log({ category, mockArticles })

	const firstArticle = mockArticles[0]

  	return (
		<div className="container py-8 lg:py-16">
			<h2 className="mb-16 text-h2">{category}</h2>

+			<div>
+				<img src={firstArticle.image} alt={firstArticle.title} />
+				<h3>{firstArticle.title}</h3>
+				<span>{firstArticle.category}</span>
+			</div>
		</div>
	)
}

```

Having to write out the `firstArticle` variable and then access its properties is a bit verbose. We can simplify this by using [**object destructuring**](https://vinodht.medium.com/javascript-object-destructing-cheat-sheet-66dedfdc412).

:::tip[Challenge]

### Simplify the code with object destructuring

    :::note

    **TIP**: When destructuring the `category` property, you will need to rename it to `articleCategory` to avoid a naming conflict with the `category` variable.

    :::

2. After destructuring each property, replace all the `firstArticle` references with direct property names.

    For example, instead of writing:

		```tsx
		<img src={firstArticle.image} alt={firstArticle.title} />
		```

		you can write:

		```tsx
		<img src={image} alt={title} />
		```

Although this does the job, it doesn't look all that great!

Let's wrap the article in a parent `div` that applies the `grid` class to organise its children. This will allow us to style the article more effectively.

```diff lang="tsx" title="app/routes/news.$category.tsx"
export default function NewsCategoryPage() {
	const { category, mockArticles } = useLoaderData<typeof loader>()

	console.log({ category, mockArticles })

	const { image, title, category: articleCategory } = mockArticles[0]

	return (
		<div className="container py-8 lg:py-16">
			<h2 className="mb-16 text-h2">{category}</h2>

+			<div className="grid grid-cols-1 gap-6 md:grid-cols-4 xl:grid-cols-3">
				<div>
					<img src={image} alt={title} />
					<h3>{title}</h3>

					<span>{articleCategory}</span>
				</div>
+			</div>
		</div>
	)
}
```

This will give us a more organised layout for our article:
