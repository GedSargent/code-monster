## Keyboard shortcuts

### Code editing

| Command | Windows | Mac |
| ----------- | ----------- | ----------- |
|  Organize imports | <MonsterKbd>Shift + Alt + O</MonsterKbd> | <MonsterKbd>Shift + Option + O</MonsterKbd> |
| Move current line up/down | <MonsterKbd>Alt + Up/Down</MonsterKbd> | <MonsterKbd>Option + Up/Down</MonsterKbd> |
| Duplicate current line above/below | <MonsterKbd>Shift + Alt + Up/Down</MonsterKbd> | <MonsterKbd>Shift + Option + Up/Down</MonsterKbd> |
| Delete current line | <MonsterKbd>Ctrl + Shift + K</MonsterKbd> | <MonsterKbd>Cmd + Shift + K</MonsterKbd> |

### File navigation

| Command | Windows | Mac |
| ----------- | ----------- | ----------- |
|  Open file | <MonsterKbd>Ctrl + P</MonsterKbd> | <MonsterKbd>Cmd + P</MonsterKbd> |
| Jump to definition | <MonsterKbd>Ctrl + Click</MonsterKbd> | <MonsterKbd>Cmd + Click</MonsterKbd> |
| Find in current file | <MonsterKbd>Ctrl + F</MonsterKbd> | <MonsterKbd>Cmd + F</MonsterKbd> |
| Find in project | <MonsterKbd>Ctrl + Shift + F</MonsterKbd> | <MonsterKbd>Cmd + Shift + F</MonsterKbd> |

### Terminal
| Command | Windows | Mac |
| ----------- | ----------- | ----------- |
| Stop server (in terminal) | <MonsterKbd>Ctrl + C</MonsterKbd> | <MonsterKbd>Cmd + C</MonsterKbd> |
| New terminal | <MonsterKbd>Ctrl + Shift + '</MonsterKbd> | <MonsterKbd>Cmd + Shift + `</MonsterKbd> |
